# Дизайн-система Александра Кандеева

Единый визуальный язык для сайтов, лендингов, презентаций, соцсетей и документов.
Премиум-тёмный регистр: монохром, слоновая кость, бронзовый акцент, крупная
типографика и фотография. Якорь стиля - whitewill.ru.

**Живая витрина:** https://arkandeev.github.io/design-system/

---

## Быстрый старт

Подключи токены и базовый CSS - и пиши разметку готовыми классами:

```html
<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/arkandeev/design-system/core/tokens.css">
<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/arkandeev/design-system/web/base.css">

<a class="btn btn--solid btn--arrow">Обсудить сессию <span class="arr">›</span></a>
<section class="theme-emerald"> … блок целиком в изумрудной теме … </section>
```

Локально - те же файлы из `core/tokens.css` и `web/base.css`.
Шаблон страницы целиком - `web/starter.html`.

## Что внутри

| Папка | Что |
| --- | --- |
| `core/` | **СПЕЦИФИКАЦИЯ** (шрифты, размеры, цвета, фоны), токены `tokens.css`, доки по типографике, цвету, темам, тёмному/светлому, движению, изображениям, логотипу |
| `web/` | рабочий `base.css`, компоненты, готовые шаблоны и демо-страницы |
| `brand/` | сгенерированные рендеры (`illustrations/`), примеры фото (`sample-photos/`), место под логотип и Futura PT |
| `presentations/` `social/` `documents/` | правила под слайды, Telegram, раздатку |
| `references/` | разбор референсов - почему именно так |
| `_build/` | сборка одностраничника и публикатор на GitHub Pages |

Начни с [`core/спецификация.md`](core/спецификация.md) и [`INDEX.md`](INDEX.md).

## Главное в двух строках

- **Дорого = сдержанность.** Почти-чёрный фон, один приглушённый акцент (бронза
  `#BF9E77`), много воздуха, острые углы. Цвет несёт смысл, а не украшает.
- **Шрифты:** заголовки Montserrat (прописными), текст Inter. Апгрейд до Futura PT
  по лицензии - одной правкой.

## Использование и развитие

- Код (токены, CSS, компоненты) - бери, адаптируй, форкай, присылай Pull Request.
- Фотографии и имя Александра Кандеева - его личный бренд, не для выдачи за свой.
- Идеи и правки - через Issues/PR.

## Демо-страницы

- Витрина-одностраничник - корень (`index.html`).
- Лендинг с чередованием тёмного/светлого - `web/site-light-dark.html`.
- Кнопки и состояния - `web/showcase-buttons.html`.
- Цвет в деле (плашки, теги, темы) - `web/colors-in-use.html`.
- Блоки с рендерами - `web/site-render.html`.
