# Презентации

Тот же премиум-тёмный язык на слайдах. Для `.pptx` - через скилл `pptx`
(инструментарий: node через nvm, soffice, рендер PyMuPDF - см. память проекта).

## Базис слайда

- **Формат** 16:9. Фон - `#0D0D0D` по умолчанию. Светлые слайды-исключения -
  на слоновой кости `#F0EAE3` с тёмным текстом.
- **Поля** крупные: не меньше 7-8% ширины с каждой стороны. Воздух важнее, чем
  уместить больше.
- **Сетка** - левый верх для надзаголовка, низ/центр для главного. Как на сайте.

## Типографика на слайдах

- Заголовки - **Montserrat прописными**, очень крупно (один слайд - одна мысль).
- Текст - Inter, мало. Слайд не документ: тезис, не абзац.
- Подписи и нумерация - прописные с разрядкой, бронза точечно.

## Типы слайдов

1. **Обложка** - фулл-блид тёмное фото + гигантский прописной заголовок снизу +
   монограмма АК в углу + бронзовая печать. Прямой перенос hero сайта.
2. **Раздел** - крупный номер и название раздела по центру, волосяная линия.
3. **Контент** - надзаголовок + заголовок + 2-3 тезиса или 3 колонки (как `.steps`).
4. **Цифры** - крупные метрики в ряд с линиями-разделителями, одно число бронзой.
5. **Кейс** - ч/б фото слева, цитата клиента справа, атрибуция прописными.
6. **Финал** - имя, контакт, печать. Минимум.

## Графики и схемы

- Палитра графиков - монохром слоновой кости разной прозрачности + **один**
  бронзовый ряд для ключевого. Не радуга.
- Линии тонкие, без теней и объёма. Подписи прописные мелкие.
- Сетка и оси - `--c-line` (почти невидимые).

## Что удешевляет (не делать)

- Цветные плашки, градиентные заливки текста, тени у фигур, иконки-клипарт.
- Больше одной мысли на слайд. Мелкий текст «чтобы влезло».
